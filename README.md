# workflow-editor

## Summary

javascript-table is a versatile React package designed to build forms with validation in easy and simple way.

## Key Features

- Simple Form
- Multistep Forms
- Form Validation
- Works with React.js

## Example

```
import { WorkflowEditor } from "wizard-editor";

class App extends React.Component {
  constructor(props) {
    super(props);
    this.state = {};
  }
  render() {
    return (
      <div>
        <WorkflowEditor />
      </div>
    );
  }
}

export default App;
```
